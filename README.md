# Fake Store App

This is a demo e-commerce application built with React, Vite, React Bootstrap, and FakeStoreAPI.

## Features

- Home page with navigation
- Product listing with FakeStoreAPI integration
- Product details with CRUD operations
- Add/Edit product forms
- Navigation bar with mobile responsiveness
- Enhanced UX: loading states, error handling, toast notifications

## Tech Stack

- React 19
- Vite
- React Bootstrap & Bootstrap 5
- Axios
- React Router DOM

## How to Run

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open [http://localhost:5173](http://localhost:5173) in your browser

## API Disclaimer

This app uses [FakeStoreAPI](https://fakestoreapi.com/). Create, update, and delete operations are for demo purposes only and do not persist data.
